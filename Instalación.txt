1. Clonar el proyecto el cual contiene todos los archivos necesarios para su ejecución (No es necesario ejecutar comandos de composer)
2. Importar el sql que se encuentra en la raiz del proyecto (homeselect_test.sql)
3. Configurar la base de datos en config/db.php de acuerdo a los datos mysql de su computador.
4. Una vez ya se tiene importada y configurada la base de datos, se puede usar la aplicación.
5. En los requerimientos de la prueba, se encontraba cargar unas categorías por base de datos, pero
   se realizó un módulo CRUD para las categorías, por lo que se pueden agregar o eliminar las que desee
   sin tener que cargarlas por base de datos.
